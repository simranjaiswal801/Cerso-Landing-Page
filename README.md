# 🌐 Cerso Landing Page

A modern, responsive, and professional landing page developed as part of the **Cerso Web Development Internship Task**.

The website is designed to showcase Cerso's services, build trust with potential clients, and provide an easy way to request a quote.

## ✨ Features

- 🎨 Modern & Professional UI
- 📱 Fully Responsive Design
- 🧭 Sticky Navigation Bar
- 🍔 Mobile Hamburger Menu
- 🚀 Hero Section with CTA
- 👨‍💼 About Cerso
- 💻 Services Section
- ⭐ Why Choose Cerso
- 🔄 Process Timeline
- 💬 Testimonials
- 📩 Contact / Quote Form
- ⬆️ Back-to-Top Button
- ✨ Smooth Scrolling & Hover Effects
- 🎯 Clean Layout & User-Friendly Experience

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Font Awesome
- Google Fonts (Poppins)
- AOS (Animate On Scroll)
